# 🍔 Delicious Burger

Um site moderno e **100% responsivo** para uma hamburgueria gourmet fictícia, **Delicious Burger**.  
Este projeto demonstra uma experiência de usuário completa e otimizada para todos os dispositivos, desde smartphones até desktops.

---

## 🚀 Demonstração ao Vivo
Confira a versão ao vivo do projeto aqui:  
👉 [Delicious Burger - Deploy na Vercel](https://projeto-hamburgueria-pi.vercel.app/)

## ✨ Funcionalidades

### 🎯 **Design 100% Responsivo**
- **Mobile-First**: Desenvolvido com abordagem mobile-first
- **Breakpoints Otimizados**: 
  - Mobile Small (320px - 480px)
  - Mobile Large (481px - 767px) 
  - Tablet (768px - 1023px)
  - Desktop (1024px - 1199px)
  - Large Desktop (1200px+)
- **Orientação Landscape**: Otimizado para dispositivos em modo paisagem
- **Touch-Friendly**: Elementos interativos com tamanho mínimo de 44px

### 🛒 **Carrinho de Compras Interativo**
- Adicione itens ao carrinho
- Atualize quantidades
- Remova itens
- Visualize o preço total
- Estado do carrinho persistente com `localStorage`
- **Modal responsivo** que se adapta a todos os tamanhos de tela

### 🔍 **Busca em Tempo Real**
- Pesquisa instantânea por nome, descrição ou ingredientes
- **Interface otimizada para mobile** com teclado virtual
- Suporte a navegação por teclado (Enter para buscar)

### 📱 **Navegação Mobile Otimizada**
- **Menu hambúrguer** com animações suaves
- **Swipe gestures** para fechar o menu
- **Focus management** para acessibilidade
- **Overlay de busca** que fecha automaticamente o menu mobile

### 🎨 **UI/UX Moderno e Acessível**
- Design limpo com transições suaves
- **Notificações responsivas** que se adaptam ao tamanho da tela
- **Contraste otimizado** seguindo padrões WCAG
- **Suporte a dark mode** (prefers-color-scheme)
- **Reduced motion** para usuários com preferências de acessibilidade

### 📄 **Layout Multi-página Responsivo**
- Início com banner hero adaptativo
- Cardápio com grid responsivo
- Sobre com layout flexível
- Contato com formulário otimizado
- Política de privacidade com texto legível

### ♿ **Acessibilidade Avançada**
- **Skip links** para navegação por teclado
- **ARIA labels** em todos os elementos interativos
- **Screen reader** friendly
- **Keyboard navigation** completa
- **Focus indicators** visíveis

## 📱 **Responsividade Detalhada**

### **Mobile Small (320px - 480px)**
- Header compacto com ícones menores
- Banner hero com altura reduzida (250px)
- Cards em coluna única
- Menu mobile em tela cheia
- Botões com tamanho otimizado para touch

### **Mobile Large (481px - 767px)**
- Layout intermediário entre mobile e tablet
- Grid de cards adaptativo
- Formulários com espaçamento otimizado
- Modal de carrinho em tela cheia

### **Tablet (768px - 1023px)**
- Menu desktop oculto, menu mobile ativo
- Grid de 2 colunas para cards de contato
- Banner hero com altura média (450px)
- Modal de carrinho com largura limitada

### **Desktop (1024px+)**
- Menu desktop completo
- Grid de 3+ colunas
- Banner hero em tamanho completo (550px)
- Layout otimizado para mouse e teclado

## 🏁 Como Começar

### Pré-requisitos
- Apenas um navegador moderno é necessário
- Para melhor experiência, recomenda-se usar um **Live Server** no editor de código

### Instalação
1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador
3. Ou use um servidor local para desenvolvimento

### Testando Responsividade
- Use as **DevTools** do navegador para testar diferentes resoluções
- Teste em dispositivos reais para melhor experiência
- Verifique a acessibilidade com **screen readers**
